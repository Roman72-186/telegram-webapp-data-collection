# Инструкция по тестированию — Сбор данных

## Подготовка к тестированию

1. Убедитесь, что у вас есть доступ к аккаунту Leadteh (rb257034.leadteh.ru)
2. Убедитесь, что webhook `fe02b6a2-14d6-46c3-836f-5fb8292d0e4f` настроен в Leadteh
3. Убедитесь, что контакт с соответствующим telegram_id существует в Leadteh (пользователь написал боту `/start`)

## Тестирование интеграции

1. Откройте веб-приложение через Telegram бота
2. Заполните форму с корректными данными:
   - Вес (кг) — положительное число, например 70.5
   - Грудь (см) — положительное число, например 90
   - Пупок (см) — положительное число, например 80
   - Живот (см) — положительное число, например 85
   - Бёдра (см) — положительное число, например 95
   - Ноги (см) — положительное число, например 55
   - Руки (см) — положительное число, например 30
3. Опционально загрузите до 3 фото (проверьте превью)
4. Нажмите кнопку «Отправить замеры»
5. После успешной отправки форма скроется и появится сообщение об успехе
6. Проверьте в Leadteh, что данные были получены

## Проверка результата в Leadteh

1. Войдите в аккаунт rb257034.leadteh.ru
2. Найдите контакт по telegram_id
3. Проверьте переменные:
   - weight, chest, navel, stomach, hips, legs, arms
   - photo_1, photo_2, photo_3 (если загружались)
   - telegram_id
   - source (должно быть `telegram-webapp-data-collection`)
   - submission_date

## Тестирование валидации

1. **Пустые поля:** оставьте все поля пустыми и нажмите «Отправить» — должны появиться ошибки у каждого поля
2. **Нечисловые значения:** введите буквы — поле не должно принять (type="number")
3. **Ноль или отрицательное:** введите 0 или -5 — должна появиться ошибка
4. **Без фото:** отправьте только замеры без фото — должно работать
5. **С фото:** загрузите фото, проверьте превью и отправьте

## Тестирование фото

1. Нажмите на область загрузки фото
2. Выберите изображение
3. Проверьте, что превью отображается
4. Проверьте в Network tab, что base64 строка присутствует в payload

## Возможные ошибки и решения

1. **Данные не доходят до Leadteh:**
   - Проверьте, что контакт с telegram_id существует
   - Проверьте, что webhook URL правильный
   - Посмотрите логи в Vercel Dashboard → Functions

2. **Форма не отправляется:**
   - Откройте консоль браузера (F12) и проверьте ошибки
   - Убедитесь, что все 7 полей замеров заполнены корректными числами

3. **Ошибка 502 (Leadteh webhook error):**
   - Webhook может быть отключён в Leadteh
   - Проверьте URL webhook в `api/submit.js`

4. **Фото не загружается:**
   - Проверьте, что файл является изображением
   - Проверьте консоль браузера на ошибки сжатия

## Тестирование без Telegram

Для тестирования без Telegram можно отправить запрос напрямую:

```bash
curl -X POST https://YOUR-VERCEL-URL/api/submit \
  -H "Content-Type: application/json" \
  -d '{"weight":70.5,"chest":90,"navel":80,"stomach":85,"hips":95,"legs":55,"arms":30,"telegram_id":123456789}'
```
